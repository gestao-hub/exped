# npm audit residual - 2026-07-14

## Escopo

Auditoria de dependencias de producao executada com:

```sh
npm audit --omit=dev --json
```

Nenhum `npm audit fix --force` foi usado. O Next.js permanece em `16.2.6`.

## Resultado

Antes da atualizacao, o npm reportava cinco entradas no grafo de producao:

- `@babel/core` (`GHSA-4x5r-pxfx-6jf8`), severidade baixa;
- `hono`, severidade alta no agregado das recomendacoes vigentes;
- `js-yaml` (`GHSA-h67p-54hq-rp68`), severidade moderada;
- `next` e seu `postcss` embutido (`GHSA-qx2v-qp2m-jg93`), severidade moderada.

Os overrides exatos atualizaram `@babel/core` para `7.29.7`, `esbuild` para
`0.28.1`, `hono` para `4.12.30` e `js-yaml` para `4.3.0`. `vitest` e
`@vitest/ui` foram atualizados em conjunto para `4.1.10`. Depois da regeneracao
do lockfile, as entradas de `@babel/core`, `esbuild`, `hono` e `js-yaml`
deixaram de aparecer na auditoria.

## Risco residual aceito

O resultado posterior ainda contem duas entradas de grafo, `next` e `postcss`,
causadas por uma unica recomendacao:

- `next@16.2.6` inclui `postcss@8.4.31`;
- a recomendacao `GHSA-qx2v-qp2m-jg93` afeta versoes de PostCSS anteriores a
  `8.5.10` e descreve XSS ao serializar CSS com `</style>` nao escapado;
- o reparo sugerido pelo npm e `next@9.3.3`, um downgrade principal
  incompativel e, portanto, rejeitado.

Referencia: https://github.com/advisories/GHSA-qx2v-qp2m-jg93

Esse risco deve ser reavaliado quando uma versao compativel do Next.js usada
pelo projeto atualizar o PostCSS embutido. Ate la, a aplicacao nao deve tratar
CSS fornecido por usuarios como conteudo confiavel nem inserir sua serializacao
diretamente em elementos `style`.

## Evidencias de verificacao

- baseline: 5 entradas (`1 low`, `3 moderate`, `1 high`);
- depois: 2 entradas (`2 moderate`), ambas no mesmo caminho Next/PostCSS;
- auditoria completa, incluindo ferramentas de desenvolvimento: as mesmas 2
  entradas; o achado baixo de `esbuild` foi removido com `0.28.1`;
- Next.js permaneceu em `16.2.6`;
- nenhuma correcao forcada ou downgrade foi aplicado.

Os testes completos, typecheck e build de producao sao gates obrigatorios antes
da promocao desta alteracao.
